# ⌨️ Typing Speed Test Game

Test your typing speed and accuracy using this fun game built with HTML, CSS, and JavaScript.

## 🚀 Features
- Random quotes to type
- Real-time WPM calculation
- Accuracy tracking
- Timer starts on first keypress
- Restart button

## 🛠️ Tech Stack
- HTML5
- CSS3
- Vanilla JavaScript
